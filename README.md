# App de Lista de contactos(Front-end)
Aplicación que permite crear, actualizar, buscar y eliminar contactos y sus números telefónicos.
Interfaz web que consume la API REST desarrollada en el repositorio back-end.

## Back-end
El back-end de este proyecto se encuentra en:
https://github.com/yanpieregc/phonebook-app-backend

## Tecnologías
- JavaScript
- React

## Características
- Listar contactos
- Buscar contactos por nombre
- Agregar nuevos contactos
- Actualizar el número telefónico de un contacto existente
- Eliminar contactos

## Instalación
- pnpm install
- pnpm run dev
